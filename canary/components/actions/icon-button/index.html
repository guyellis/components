IconButton

STABLE

IconButton's are used when you need an icon to trigger an action, such as showing a dialog or deleting an item. It require a label that describes it's action. The label should be descriptive and succinct. If you need an icon for purely decorative purposes use an <Icon /> instead.

<Space evenly>
  <IconButton icon={<Add />} label="Add Something Neat" />
  <IconButton icon={<Settings />} label="Settings" />
  <IconButton icon={<Delete />} label="Trash It" />
  <IconButton icon={<Close />} label="Close" />
  <IconButton icon={<Favorite />} label="Add to Favorites" />
</Space>

IconButton Options

Sizes

  • xxsmall | xsmall (default) | small | medium | large
<Space evenly>
  <IconButton icon={<Favorite />} label="Add to Favorites" size="xxsmall" />
  <IconButton icon={<Favorite />} label="Add to Favorites" />
  <IconButton icon={<Favorite />} label="Add to Favorites" size="small" />
  <IconButton icon={<Favorite />} label="Add to Favorites" size="medium" />
  <IconButton icon={<Favorite />} label="Add to Favorites" size="large" />
</Space>

Toggle

Icon buttons have the toggle prop which can be used to show a toggleable state. The toggle prop has to be explictly set to either true or false.

;() => {
  const [isToggled, setIsToggled] = useState(false)
  return (
    <IconButton
      icon={isToggled ? <Favorite /> : <FavoriteOutline />}
      toggle={isToggled}
      onClick={() => setIsToggled(!isToggled)}
    />
  )
}

ToggleBackground

Icon buttons have the toggleBackground prop which can be used to show a background color to the Icon. The toggle prop has to be explictly set to either true or false.

<Space evenly>
  <IconButton icon={<Favorite />} toggle toggleBackground />
  <IconButton icon={<Favorite />} shape="round" toggle toggleBackground />
</Space>

ToggleColor

Icon buttons have the toggleColor prop which can be used to specify what the icon's color and background color are when toggle = true. It accepts the following values:

  • 'key'
  • 'calculation'
  • 'dimension'
  • 'measure'
<Space evenly>
  <IconButton
    icon={<Favorite />}
    toggle
    toggleBackground
    toggleColor="calculation"
  />
  <IconButton
    icon={<Favorite />}
    toggle
    toggleBackground
    toggleColor="calculation"
  />
</Space>

IconButton & Tooltip

You can also use your own Tooltip around IconButton (this will effectively remove the "built-in" Tooltip)

<Tooltip content="I'm a little teapot">
  <IconButton label="Add Something Neat" icon={<Add />} size="xxsmall" />
</Tooltip>

Disabling the Tooltip

If you would like to disable the tooltip entirely, use the tooltipDisabled prop.

<IconButton label="no tooltip" icon={<Home />} size="large" tooltipDisabled />

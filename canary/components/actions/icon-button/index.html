IconButton

STABLE

IconButton's are used when you need an icon to trigger an action, such as showing a dialog or deleting an item. It require a label that describes it's action. The label should be descriptive and succinct. If you need an icon for purely decorative purposes use an <Icon /> instead.

<Space evenly>
  <IconButton icon="Plus" label="Add Something Neat" />
  <IconButton icon="Gear" label="Settings" />
  <IconButton icon="Trash" label="Trash It" />
  <IconButton icon="Close" label="Close" />
  <IconButton icon="Favorite" label="Add to Favorites" />
</Space>

IconButton Options

  • xxsmall | xsmall (default) | small | medium | large

Toggle

Icon buttons have the toggle prop which can be used to show a toggleable state. The toggle prop has to be explictly set to either true or false.

;() => {
  const [isToggled, setIsToggled] = useState(false)
  return (
    <IconButton
      icon={isToggled ? 'Favorite' : 'FavoriteOutline'}
      toggle={isToggled}
      onClick={() => setIsToggled(!isToggled)}
    />
  )
}

IconButton & Tooltip

You can also use your own Tooltip around IconButton (this will effectively remove the "built-in" Tooltip)

<Tooltip content="I'm a little teapot">
  <IconButton label="Add Something Neat" icon="Plus" size="xxsmall" />
</Tooltip>

Disabling the Tooltip

If you would like to disable the tooltip entirely, use the tooltipDisabled prop.

<IconButton label="no tooltip" icon="Home" size="large" tooltipDisabled />

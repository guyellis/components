List

STABLE

List, along with its companion component ListItem, renders a list of items. However, unlike standard ul and li elements, List and ListItem come prebuilt with a number of props to simplify item style and behavior customization.

<List>
  <ListItem icon="Calendar" description="Orange-y" detail="England">
    Cheddar
  </ListItem>
  <ListItem icon="Pivot" detail="Netherlands">
    Gouda
  </ListItem>
  <ListItem icon="AnalyticsApp" detail="Italy">
    Mozzarella
  </ListItem>
  <ListItem icon="Account" detail="Switzerland">
    Swiss
  </ListItem>
</List>

Use the density prop to set the size and spacing of your List. As density decreases, so does the height and spacing of your List's child ListItems.

density values range from -3 to 1.

Density: 0
;() => {
  const [density, setDensity] = useState(0)
  return (
    <>
      {`Density: ${density}`}
      <Space gap="small" mt="small">
        <Button onClick={() => density !== 1 && setDensity(density + 1)}>
          + Density
        </Button>
        <Button onClick={() => density !== -3 && setDensity(density - 1)}>
          - Density
        </Button>
      </Space>
      <Divider />
      <List density={density}>
        <ListItem icon="Calendar">Item 1</ListItem>
        <ListItem icon="Calendar">Item 2</ListItem>
        <ListItem icon="Calendar">Item 3</ListItem>
      </List>
    </>
  )
}

Use the iconGutter prop if you want ListItem children without icons to align their text with ListItem children with icons.

<>
  <List>
    <ListItem icon="Calendar">Item 1</ListItem>
    <ListItem>Item 2</ListItem>
    <ListItem>Item 3</ListItem>
  </List>
  <Divider />
  <List iconGutter>
    <ListItem icon="Calendar">Item 1</ListItem>
    <ListItem>Item 2</ListItem>
    <ListItem>Item 3</ListItem>
  </List>
</>

ListItem

ListItem renders an li element and should be the child of a List element.

The table below displays ListItem's prop names and descriptions.

PropType ValuesDescription
colorstringIf the child element is of type string, color determines the color of text and icon, if present.
descriptionReactNodedescription is an optional element that sits underneath the child element.
detailReactNodedetail is an optional element placed right of the child element.
iconIconName (See Icon for valid icon names)icon is an optional icon element placed left of the child element.
iconArtworkReactNodeiconArtwork allows you to use an svg instead of an icon name string to render a custom icon.
itemRole"link" | "button"itemRole sets the correct accessible role for the item. Below are its possible values:
  • Use "link" for items that navigate to another page.
  • Use "button" for items that trigger in-page interactions, like displaying a dialog.
truncatebooleanIf true and text overflows, text children and description will be truncated.
keyColorbooleanUsing keyColor will replace the normal ui2 background color for selected and selected + hovered items with key colors.
disabledbooleanIf true, the item will have the following properties:
  • color: text1
  • cursor: not-allowed
  • hover background: background (i.e. none)
selectedbooleanIf true, the item will have a ui2 background color. Also, aria-selected will be "true" for this item.
currentbooleanIf true, the item will have a ui2 background color. Also, aria-current will be "true" for this item.

Note: In addition these custom props, ListItem also accepts standard handlers like onClick, onKeyDown, etc.

    • <ListItem
      color="orange"
      icon="Calendar"
      description="Orange-y"
      detail="England"
      onClick={() => alert('Get that Cheddar')}
    >
      Cheddar
    </ListItem>

    Customizing a ListItem detail

    You can customize the look and behavior of detail elements by providing the detail prop an object with an "options" property.

    Enabling the hoverDisclosure option will hide and show the detail on hover on and hover off.

    Enabling the accessory option will render the detail element outside of the label container. This option is most commonly used when you'd like to have a button as your detail element.

    • <List>
      <ListItem
        icon="FieldNumber"
        detail={
          <IconButton
            icon="Pivot"
            label="Pivot"
            onClick={() => alert("You've pivoted!")}
          />
        }
      >
        ListItem (Standard)
      </ListItem>
      <ListItem
        icon="FieldNumber"
        detail={{
          content: (
            <IconButton
              icon="Pivot"
              label="Pivot"
              onClick={() => alert("You've pivoted!")}
            />
          ),
          options: {
            accessory: true,
            hoverDisclosure: true,
          },
        }}
      >
        ListItem (Options Enabled)
      </ListItem>
    </List>
